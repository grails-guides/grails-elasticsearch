You can create a domain class with the create-domain-class CLI command from the root of your project:

[source, bash]
----
$ ./grailsw create-domain-class Book
| Created grails-app/domain/demo/Book.groovy
| Created src/test/groovy/demo/BookSpec.groovy
----

Define following properties in the domain class:

[source, groovy]
.grails-app/domain/demo/Book.groovy
----
include::{sourceDir}/grails-app/domain/demo/Book.groovy[lines=1..15]
include::{sourceDir}/grails-app/domain/demo/Book.groovy[lines=21..28]
----

Generate scaffold service, controller and json-views  for `Book` domain by running following bash command:

[source,bash]
----
./grailsw generate-all demo.Book
